# Description

Elevator Emulator with localStorage

- Calling the elevator from different floors
- Saving state in localStorage

# Technologies:

- Vue
- Typescript
- Sass
- Eslint, prettier

# To start the project, you need to run next commands:

- `npm i`
- `npm run serve`
